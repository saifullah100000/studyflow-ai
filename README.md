# StudyFlow AI

StudyFlow AI is an AI-powered learning platform that generates personalized notes, quizzes and flashcards, converts them into professional PDFs and sends the PDFs through WhatsApp.

## Technology Stack

- React
- TypeScript
- Tailwind CSS
- NestJS
- PostgreSQL
- Prisma
- n8n
- Google Gemini
- Puppeteer
- Cloudflare R2
- WhatsApp Cloud API
- Docker
