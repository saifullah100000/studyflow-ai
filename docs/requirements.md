# StudyFlow AI Requirements

## Project Goal

StudyFlow AI generates personalized study notes, quizzes and flashcards. It converts the generated content into a PDF and delivers the PDF through WhatsApp.

## User Features

- User registration and login
- Generate notes from a topic
- Select difficulty level
- Select language
- Select notes length
- Generate MCQs
- Generate flashcards
- Preview and edit notes
- Download notes as PDF
- Send PDF through WhatsApp
- View generation history

## Admin Features

- View users
- View generation jobs
- View failed workflows
- View delivery logs
- Disable user accounts
- Set generation limits

## Main Workflow

Topic → AI Generation → Store Notes → Generate PDF → Upload PDF → Send through WhatsApp

## First-Version Limitations

- No payment system
- No mobile application
- No OCR
- No real-time collaboration
- Gemini will be the only AI provider
